3. Customization Guide üé®

3.1. Navigation and Active State

To highlight the current page in the navbar, use the Bootstrap classes active and aria-current="page" on the appropriate <a> tag.

<!-- Example: The 'Shop' link is active on shop.html -->
<li class="nav-item">
    <a class="nav-link active" aria-current="page" href="shop.html">Shop</a>
</li>

The **Active State** style (dark underline) is defined in css/style.css and can be customized there.

3.2. Search Modal

The search icon does not link to a dedicated page. Instead, it triggers a **Bootstrap Modal** for an optimal user experience.

The modal HTML is located near the bottom of each HTML file, just before the closing </body> tag.

3.3. Favicon (Site Icon)

The template is configured to use a favicon. Simply replace the placeholder file:

  • **File to Replace:** /favicon.png (must be placed in the root folder, outside html/).
  • The link tag is already included in the <head> of all HTML files: <link rel="icon" type="image/png" href="../favicon.png">

3.4. Footer Links Visibility Fix

The issue where footer links were invisible on the dark background has been corrected. The links now use text-white text-opacity-75 for high contrast:

<ul class="list-unstyled text-small">
    <li><a href="about.html" class="text-white text-opacity-75 text-decoration-none">About Us</a></li>
</ul>

4. Dependencies

The following external resources are used and linked via CDN in the <head> section of every page:

  • Bootstrap 5.3: https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css
  • Font Awesome 6: https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css (For all icons)
  • Google Fonts (Poppins): https://fonts.googleapis.com/css2?family=Poppins...

The core JavaScript library is loaded at the end of the <body> for faster page loading:

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
